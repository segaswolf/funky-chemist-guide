# 🧪 Funky Chemist Guide

A small, funky, and practical chemistry guide made as a simple HTML page.

This project is meant to be a lightweight reference for chemical recipes, mixing logic, and quick consultation while playing or learning chemistry-related systems.

## 🌐 Live Page

Once GitHub Pages is enabled, the guide should be available here:

https://segaswolf.github.io/funky-chemist-guide/

## ✨ Features

- Simple static HTML page
- Lightweight and easy to open
- No installation required
- Useful as a quick reference guide
- Designed to be expanded over time
